# vayuayan Notebooks

Jupyter notebooks demonstrating vayuayan package usage.

## Notebooks

- `01_getting_started.ipynb` - Basic package usage
- `02_historical_data_analysis.ipynb` - Historical AQI data analysis
- `03_live_monitoring.ipynb` - Real-time air quality monitoring
- `04_pm25_regional_analysis.ipynb` - PM2.5 regional analysis with GeoJSON

Start with `01_getting_started.ipynb` for an introduction.
